## Commands

### `build`

Builds your Noderize app.

### `watch`

Builds and starts your Noderize app as you code. Used for development.

### `test`

Runs tests (Jest).

### `format`

Runs code formatter (Prettier).

### `start`

Start the built file (after running `build` command).
