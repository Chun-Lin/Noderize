Configuration: Jest

You may configure Jest as you normally would with a slight addition:

The Jest configuration is loaded using cosmiconfig, meaning you can use the normal Jest ways (jest key in package.json or jest.config.js), plus you can use a .jestrc file (with optional .json/.yml/.yaml/.js extension) for configuring using JSON/YAML/JS.

To pass arguments to Jest when using the test script, simply add them after -- -- like so:

yarn test -- -- --showConfig
# or
npm test -- -- --showConfig
